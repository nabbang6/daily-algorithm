## 💡 문자열 압축

알파벳 대문자로 이루어진 문자열을 입력받아 같은 문자가 연속으로 반복되는 경우 반복되는
문자 바로 오른쪽에 반복 횟수를 표기하는 방법으로 문자열을 압축하는 프로그램을 작성하시
오. 단 반복횟수가 1인 경우 생략합니다.

### 입력 설명

첫 줄에 문자열이 주어진다. 문자열의 길이는 100을 넘지 않는다.

### 출력 설명

첫 줄에 압축된 문자열을 출력한다.

### 입력 예제 1

KKHSSSSSSSE

### 출력 예제 1

K2HS7E

## 📌 풀이 방법

인접 문자 비교로 연속 구간을 감지하고, 구간이 끝나는 시점에 결과를 누적한다.

1. `count = 1`로 초기화한 뒤 문자열을 순회한다.
2. **현재 문자 === 다음 문자**이면 `count`를 증가시킨다.
3. **현재 문자 !== 다음 문자**이면 (연속 구간의 끝) 결과에 추가한다.
   - `count === 1`이면 문자만 추가 (숫자 생략 조건)
   - `count > 1`이면 문자 + count 추가
   - `count`를 1로 초기화한다.
4. 순회가 끝나면 결과 문자열을 반환한다.
