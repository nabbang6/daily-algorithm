## 💡 순차 연산 적용기

초기값 `x`, 보조값 `y`, 가변 인수로 연산자 목록을 받아 순서대로 적용하는 `doAll(x, y, ...ops)` 함수를 작성하세요. 함수가 아닌 인수는 무시합니다.

### 입력 예제

```js
const add = (a, b) => a + b;
const sub = (a, b) => a - b;
const pow = (a, b) => a ** b;

doAll(2, 3, add, pow, sub, 'hello');
```

### 출력 예제

```
122
```

> `add(2,3)=5` → `pow(5,3)=125` → `sub(125,3)=122` / `'hello'`는 무시

## 📌 풀이 방법

- `...ops`로 연산자 목록을 가변 인수로 받아 `for...of`로 순회한다.
- `typeof op === 'function'`으로 함수 여부를 검사해 함수가 아닌 값은 건너뛴다.
- 각 단계의 결과가 다음 연산의 첫 번째 인수가 되고, `y`는 고정값으로 재사용된다.
