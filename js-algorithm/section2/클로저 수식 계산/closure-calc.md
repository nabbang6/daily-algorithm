## 💡 클로저 수식 계산

두 수 `x`, `y`를 받아 내부에 `square(n)`, `sum()`, `diff()` 함수를 클로저로 정의하고, `square(sum()) - square(diff())`를 반환하는 `calcExpression(x, y)`를 작성하세요.

### 입력 예제

```js
calcExpression(5, 3)
```

### 출력 예제

```
60
```

> `(5+3)² - (5-3)² = 64 - 4 = 60`

## 📌 풀이 방법

- `square`, `sum`, `diff`는 외부 함수의 `x`, `y`를 참조하는 클로저 함수다.
- `sum()`과 `diff()`는 매개변수 없이 외부 스코프의 `x`, `y`를 직접 사용한다.
- `(x+y)² - (x-y)²`는 인수분해하면 `4xy`와 동일하다 (`4 × 5 × 3 = 60`).
