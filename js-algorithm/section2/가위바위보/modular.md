# 모듈러 연산 (Modular Arithmetic) 정리

## 기본 개념

`a % n`은 `a`를 `n`으로 나눈 **나머지**를 반환한다.

```
7 % 3 = 1   (7 = 3×2 + 1)
9 % 3 = 0   (9 = 3×3 + 0)
5 % 2 = 1   (5 = 2×2 + 1)
```

---

## 핵심 성질: 순환 구조

모듈러 연산의 가장 강력한 활용은 **순환(cycle)**을 표현할 때다.

```
0 % 3 = 0
1 % 3 = 1
2 % 3 = 2
3 % 3 = 0  ← 다시 0으로 돌아옴
4 % 3 = 1
5 % 3 = 2
6 % 3 = 0  ← 또 다시 0
```

`% n`을 사용하면 어떤 숫자든 `0 ~ n-1` 범위 안에서 **순환**하게 만들 수 있다.

## 가위바위보에 적용

### 문제 구조

가위바위보는 순환 구조다.

```
가위(1) → 보(3) 이김
바위(2) → 가위(1) 이김
보(3)   → 바위(2) 이김
```

이 관계를 수식으로 표현하면:

> **A가 이기는 조건: `arr2[i] === arr1[i] % 3 + 1`**

| arr1 (A) | arr1 % 3 | arr1 % 3 + 1 | arr2 (B, 지는 쪽) | 검증 |
| :------: | :------: | :----------: | :---------------: | :--: |
| 1 (가위) |    1     |      2       |      3 (보)       |  ✓   |
| 2 (바위) |    2     |      3       |     1 (가위)      |  ✓   |
|  3 (보)  |    0     |      1       |     2 (바위)      |  ✓   |

### 차이를 이용한 방법

두 값의 차이에 모듈러를 적용해서 승/무/패를 한 번에 판별할 수도 있다.

```js
(arr1[i] - arr2[i] + 3) % 3;
```

`+3`을 하는 이유: 차이가 음수가 될 수 있어서 나머지가 음수가 되는 것을 방지한다.

|  A  |  B  | A - B | (A - B + 3) % 3 |   결과   |
| :-: | :-: | :---: | :-------------: | :------: |
|  1  |  1  |   0   |        0        | 무승부 D |
|  1  |  3  |  -2   |        1        | A 승리 A |
|  2  |  1  |   1   |        1        | A 승리 A |
|  3  |  2  |   1   |        1        | A 승리 A |
|  3  |  1  |   2   |        2        | B 승리 B |
|  1  |  2  |  -1   |        2        | B 승리 B |
|  2  |  3  |  -1   |        2        | B 승리 B |

결론: **0 → D, 1 → A, 2 → B** 로 대응시키면 된다.

```js
function solution(s, arr1, arr2) {
  const result = ['D', 'A', 'B'];
  return Array.from({ length: s }, (_, i) => {
    return result[(arr1[i] - arr2[i] + 3) % 3];
  });
}
```

## 자주 쓰이는 패턴

### 1. 배열 인덱스 순환

```js
// 길이 n인 배열에서 다음 인덱스로 이동 (끝에 도달하면 처음으로)
const next = (i + 1) % n;

// 이전 인덱스 (0에서 뒤로 가면 n-1)
const prev = (i - 1 + n) % n;
```

### 2. 홀짝 판별

```js
n % 2 === 0; // 짝수
n % 2 === 1; // 홀수
```

### 3. 특정 주기마다 반복

```js
// 3의 배수인 인덱스마다 처리
if (i % 3 === 0) { ... }
```

### 4. 큰 수의 나머지 (오버플로 방지)

```js
// 값이 매우 클 때 나머지만 유지하며 계산
((a + b) % MOD(a * b)) % MOD;
```

## 요약

| 상황         | 수식              | 설명                      |
| ------------ | ----------------- | ------------------------- |
| 0 ~ n-1 순환 | `x % n`           | 범위를 벗어나지 않는 순환 |
| 음수 보정    | `(x + n) % n`     | 음수가 나오지 않게 보정   |
| 순환 비교    | `(a - b + n) % n` | 두 원소의 순환 관계 판별  |
