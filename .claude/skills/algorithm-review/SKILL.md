# Algorithm Review

사용자가 현재 파일에 작성한 알고리즘 풀이를 분석하고, 해설과 개선된 풀이를 파일 하단에 추가한다.

## Instructions

### Step 1: 현재 파일 읽기

IDE에서 열려 있는 현재 파일을 Read 도구로 읽는다.
파일을 찾을 수 없으면 사용자에게 파일 경로를 알려달라고 요청한다.

### Step 2: 문제와 풀이 파악

파일에서 다음을 파악한다:

- 주석으로 작성된 문제 설명 및 조건
- 사용자가 작성한 풀이 코드
- 사용된 프로그래밍 언어

### Step 3: 풀이 분석 및 해설 작성

파일 하단에 다음 내용을 해당 언어의 주석 형식으로 추가한다.
모든 내용은 **한국어**로 작성한다.

추가할 내용 구조:

```
/* ===== Claude 해설 =====

[풀이 분석]
- 사용자 풀이의 핵심 아이디어 설명
- 시간 복잡도: O(?)
- 공간 복잡도: O(?)
- 잘된 점 / 개선 가능한 점

[개선된 풀이] (더 나은 방법이 있을 경우)
- 개선 아이디어 설명
- 코드 예시

[핵심 개념]
- 이 문제에서 배울 수 있는 알고리즘/자료구조 개념
===== */
```

### Step 4: 파일에 내용 추가

Edit 도구를 사용해 파일 맨 끝에 위 분석 내용을 추가한다.
기존 사용자 코드는 절대 수정하지 않는다.

## 주의사항

- 사용자 코드를 절대 변경하거나 삭제하지 않는다
- 내용은 항상 파일 **하단에 추가**한다
- 주석 형식은 파일의 언어에 맞게 조정한다 (JS/Java: `/* */`, Python: `"""` 또는 `#`, 등)
- 이미 Claude 해설이 있으면 덮어쓰지 않고 사용자에게 먼저 확인한다
- 모든 설명은 **한국어**로 작성한다
